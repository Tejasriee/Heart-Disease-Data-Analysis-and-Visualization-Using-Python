# 🫀 Heart Disease Data Analysis and Visualization Using Python

## Description
This project analyzes heart disease-related health parameters to understand patient health conditions using data visualization techniques in Python.

## Project Objectives
- Analyze heart disease-related health data.
- Perform exploratory data analysis (EDA).
- Generate statistical summaries.
- Visualize health parameters using graphs.
- Identify relationships between different health parameters.

## Dataset
The dataset contains the following attributes:
- Patient ID
- Age
- Blood Pressure
- Cholesterol
- Heart Rate

## Tools and Technologies
- Python
- Google Colab
- Pandas
- Matplotlib
- Seaborn

## Project Workflow
1. Import required libraries.
2. Load the dataset.
3. Check for missing values.
4. Perform statistical analysis.
5. Calculate average values.
6. Create data visualizations:
   - Bar Chart
   - Line Graphs
   - Scatter Plot
   - Correlation Heatmap
7. Interpret the analysis results.

## Project Outcome
The project provides insights into heart disease-related health parameters through statistical analysis and data visualization. Any changes made to the dataset are immediately reflected in the analysis and graphs, demonstrating a data-driven approach.
